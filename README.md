# CrantOS
Operating system made in batch (yes you heard it right)!

## How to install it?
Just download it and run `CrantOS.bat`.

## Is CrantOS open source
As you can see it is!

## CrantOS is a virus!
No, it isn't.
Some antiviruses may detect CrantOS as a virus. Just ignore that.
